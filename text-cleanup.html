Flagship Product

Text Cleanup

AutoCAD 図面内の表記ゆれを整理する、小型の業務向けプラグインです。最初の目的は、全角英数字、半角カタカナ、括弧混在の手戻りを減らすことです。

変換例を見る 会社版を相談
主な用途 図面内文字の表記統一
対象 TEXT / MTEXT / 属性文字
標準ルール 英数字: 全角→半角 / ｶﾀｶﾅ: 半角→全角
提供形態 試用版 / 会社版 / 定制開発
Before 設備・建築・機械図面

After Plugin Result
分電盤A-102(EPS) 給気ファン(SA-1)　3φ4W 点検口(600×600)　機器記号: VAV-01

現在の標準ルール

  • 全角数字を半角へ変換
  • 全角英字を半角へ変換
  • 括弧の混在状態を整理
  • 半角カタカナを全角へ統一

会社ごとに調整できること

  • 全角 → 半角 / 半角 → 全角の方向調整
  • 部署ごとの表記基準に合わせた設定
  • 対象文字と対象図面の限定
  • 追加仕様は別見積で個別対応

会社導入時の見せ方

試用版 対象図面で効果を確認するための検証用
会社版 社内配布用のライセンスを発行
確認項目 対象文字、対象図面、運用ルール、導入台数
追加調整 変換方向や対象文字は個別調整可能

向いている場面

  • 提出前に表記ゆれをまとめて直したい
  • 社内ルールで全角 / 半角の扱いが決まっている
  • 設備記号や盤名称の見た目を揃えたい
  • 人手での文字修正に時間がかかっている

向いていない場面

  • 全文字を無条件に一括変換したい場合
  • 会社ごとの例外条件が整理されていない場合
  • 対象オブジェクトが未確認のまま導入を急ぐ場合
