AutoCAD Productivity Plugins

CAD図面作業を、小さなプラグインで確実に短縮。

文字修正、属性処理、図面チェック、社内CADルール対応。建築設備設計の現場で繰り返し発生する作業を、軽量なAutoCAD向けプラグインとして整理します。

製品機能を見る 会社版・定制開発を相談
AutoCAD向け AutoLISP中心 会社版ライセンス対応 日本語図面運用を想定
Products

今ある機能と、追加予定のプラグイン。

最初の実験対象は文字変換プラグインです。今後は同じ販売・ライセンス・検証フローで、別のCAD補助プラグインを追加できます。

検証済み

Text Cleanup

図面内の全角数字、全角英字、括弧、半角カタカナを選択範囲だけ変換します。

  • 注記・設備表記の揺れを整理
  • TEXT、MTEXT、属性文字を対象
  • 会社版ライセンスに対応
追加予定

Attribute Tools

ブロック属性、機器番号、盤名称、図面内の定型文字をまとめて確認・修正するツール群。

  • ブロック属性の抽出
  • 社内ルールに沿った表記チェック
  • 設備図面向けの一括修正
追加予定

Drawing Check

図面提出前の文字、画層、尺度、注記ルールを確認する軽量チェックツール。

  • 提出前チェックの省力化
  • チェックリストの自動化
  • 会社ごとのCAD基準に対応
Example

何をするプラグインか、すぐ分かる表示。

製品ページでは、専門用語よりも「変換前」「変換後」を前面に出します。担当者が一目で用途を判断できる構成です。

Before図面注記
ＡＢＣ１２３（テスト） 盤ＮＯ．１２（１Ｆ） ｶﾝｷﾌｧﾝ（ＥＦ－１）
AfterPlugin result
ABC123(テスト) 盤NO.12(1F) カンキファン(EF-1)
Workflow

導入は短く、確認は明確に。

小さなプラグインでも、会社で使う場合は配布、ライセンス、検証ログが必要です。導入手順を固定し、顧客側の不安を減らします。

読み込み

AutoCADにプラグインを読み込みます。

ライセンス設定

発行された会社版コードを設定します。

状態確認

製品名、有効期限、端末情報、キーを確認します。

実行

対象を選択し、必要な処理だけ実行します。

Demo

動画は60秒以内で、機能だけを見せる。

ページには動画枠を用意しています。実際のAutoCAD操作を録画し、assets/plugin-demo.mp4 として配置すれば、そのまま再生できます。

推奨構成: 読み込み → ライセンス確認 → 対象選択 → 実行 → 変換結果。説明より画面を見せる方が早いです。
