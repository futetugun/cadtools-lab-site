Products

まずは主力製品を見せ、追加製品は予告で置く。

ホームでは 1 製品だけを主役にし、他は「追加予定」か「次の候補」として短く見せる構成が B2B 向きです。詳細は個別ページへ分けます。
主力製品

Text Cleanup

図面内の全角英数字を半角へ、半角カタカナを全角へ整理します。括弧の混在整理にも対応し、選択範囲だけ処理できます。

  • 設備・建築・機械図面の文字整理
  • TEXT、MTEXT、属性文字を対象
  • 会社版ライセンスに対応
製品詳細へ
追加予定

Attribute Tools

ブロック属性、機器番号、盤名称、図面内の定型文字をまとめて確認・修正するツール群。

  • ブロック属性の抽出
  • 表記ルールの一括確認
  • 設備図面向けの補助処理
追加予定

Drawing Check

提出前の文字、層、尺度、注記ルールを確認する軽量チェックツール。

  • 提出前チェックの省力化
  • 社内ルールの見落とし防止
  • 会社ごとの CAD 基準に対応

現在の標準ルール

  • 全角数字を半角へ変換
  • 全角英字を半角へ変換
  • 括弧の混在状態を整理
  • 半角カタカナを全角へ統一

会社ごとに調整できること

  • 全角 → 半角 / 半角 → 全角の方向調整
  • 部署ごとの表記基準に合わせた設定
  • 導入前に対象文字と対象環境を確認
  • 追加仕様は別見積で個別対応
Proof
