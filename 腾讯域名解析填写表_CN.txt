二、www 子域名
主机记录: www
记录类型: CNAME
记录值: 你的 GitHub Pages 地址

示例:
如果你的 GitHub 用户名是 exampleuser
且你使用用户主页型站点，则可能填:
exampleuser.github.io

如果你使用项目型仓库，GitHub Pages 页面里会给出可用域名，优先按 GitHub 实际提示填写。

三、不要填的内容
- 不要再填腾讯云 CVM 内网地址
- 不要填没有公网 IP 的服务器
- 不要同时给 @ 配多个互相冲突的 CNAME

四、生效时间
- 一般几分钟到几小时
- 全球完全刷新有时可能到 24 小时

五、验收顺序
1. GitHub 仓库 Pages 已启用
2. 腾讯域名解析已保存
3. 打开 GitHub Pages 设置页确认自定义域名识别成功
4. 再等 HTTPS 生效
